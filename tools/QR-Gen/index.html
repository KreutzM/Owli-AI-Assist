OpenRouter QR Generator

Generate a QR code locally in your browser to import a custom OpenRouter key into the Owli app.

This page generates the QR code locally in your browser. Your key is not sent to the server.

Plain mode stores: openrouter:key=<KEY>

PIN mode stores an encrypted payload for QR transport only: openrouter:keyenc:v1:pbkdf2-sha256:...

The PIN reduces casual QR reading, but it is not strong long-term secret protection.

The key stays in memory only for this page session unless you download the QR image.

QR mode

If left blank, the default PIN is 1597. If you enter a PIN, it must be exactly 4 digits.
